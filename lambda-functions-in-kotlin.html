Lambda Functions in Kotlin

Anindra September 22, 2022

A lambda function is a small anonymous function. It is a custom make function that take some value then modify and end return some modified values. All object oriented programming language use this Lambda Function.Syntax of a lambda function like this

val lambdaName: (InputType) → ReturnType = {argument: InputType → body} 

Explain: Lambda function can be val, var both. lambdaName is a variables who take a InputType example String, Int, Boolean and return same or different type. argument is take input from lambda function that's why its type same as lambdaName input type. And body is some code where you can write your logical code, how to use this custom made lambda function. Now lets see some simple Lambda Function examples.Example 1: When you want to take one value as a parameter in a function then you can make a lambda function like this

    // when function takes only one value
    val greeter: (String) -> String = { output: String ->
        "Hello $output"
    }
    
    greeter("Anindra")

Example 2: When comes to take  more parameters then you can use lambda function like this

    // When function takes more than one value
    val anotherGreeter: (String, String) -> String = { output1: String, output2: String ->
        "Hello $output1 $output2"
    }
    
    anotherGreeter("Anindra","Karmakar")

Example 3: Parameters also can be different type like String, Int, Boolean etc, so in that case you can write a simple lambda function or lambdas

fun main(){
        val addition = {number1: Int, number2: Int ->
        println("${number1+number2}")
    }
    
    addition(23,22)
 }

In Kotlin you can skip : (InputType) → ReturnType part of code. Kotlin is that much intelligent that it can understand the input type and output type from your extra lambda code.
